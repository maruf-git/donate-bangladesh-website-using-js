Important FAQ

What is Document Object Model (DOM) ?

The Document Object Model (DOM) is a way for web browsers to represent a webpage so that programming languages like JavaScript can interact with it. The DOM represents the HTML document as a tree. Each node in the tree is an object corresponding to a part of the document, such as elements (div tag, p tag, etc.), attributes, and text.

How do you can select an element from the DOM ?

To select an element from the DOM using JavaScript, there are several methods. These methods allow us to find and interact with specific parts of the webpage.
Here are the most common ways:
1. getElementById("id")
This method selects an element by its unique id attribute.
2. getElementsByClassName("class")
This method selects all elements that share a certain class. It returns a collection (like an array).
3. getElementsByTagName("tag")
This method selects all elements by their tag name (like div, p, h1, etc.).
4. querySelector("selector")
This method selects the first element that matches a CSS-style selector. You can use id, class, or element tags.
5. querySelectorAll("selector")
This method selects all elements that match a CSS-style selector. It returns a list of all matching elements.

What is event delegation in the context of the DOM, and why is it useful ?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to handle events from its child elements. Instead of adding individual event listeners to each child, you add one listener to a common ancestor and take advantage of event bubbling (where an event propagates up the DOM tree).
Event Delegation is a useful pattern that allows us to write cleaner code, and create fewer event listeners with similar logic.With event delegation, we create fewer event listeners and perform similar events-based logic in one place. This makes it easier for us to add and remove elements without having to add new or remove existing event listeners.

2. Manipulating Styles:
We can change an element's CSS styles directly using the style property.
a. Setting Inline Styles: we can set specific CSS properties using JavaScript:
const element = document.getElementById("myElement");
element.style.color = "red"; // Changes the text color to red
b. Removing Inline Styles: To remove a style, set the property back to an empty string:
const element = document.getElementById("myElement");
element.style.color = ""; // Removes the inline color style
c. Adding Multiple Styles at Once: We can also use element.style.cssText to set multiple styles at once:
element.style.cssText = "color: green; background-color: yellow; font-size: 18px;";
